# Search

Find anything across your GitScrum workspace.

## Available Tools

| Tool | Description |
|:-----|:------------|
| `search` | Global search across all entities |

---

## Using Search

### Basic Search

```
"Search for login"
"Find tasks about authentication"
"Search 'payment integration'"
```

### What You Can Find

Search works across:
- Tasks
- User stories
- Projects
- Wiki pages
- Notes
- Comments
- Team members

## Tips

### Be Specific

More specific searches give better results:

```
"Search login bug" → Better
"Search bug" → Too broad
```

### Use Keywords

Focus on unique words:

```
"Search OAuth integration" → Good
"Search the OAuth thing" → Less effective
```

---

## Common Workflows

### Finding Related Work

```
1. "Search for authentication"
2. Review results
3. "Get details for task [found-uuid]"
```

### Avoiding Duplicates

Before creating new work:

```
1. "Search for payment feature"
2. Check if similar task exists
3. Create new or update existing
```
